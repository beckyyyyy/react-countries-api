## My process

### Function

1. 首頁可以查看所有國家清單 (一次顯示 24 筆，若需查看更多可點擊 "more")
2. 可以透過搜尋國家名字，或使用 Filter by region 查看特定國家清單。
3. 點擊任意國家，將進入詳細資訊頁。
4. 詳細資訊頁除顯示國家資訊，也顯示周圍國家，可以點擊周圍國家進入該國詳細資料頁。
5. 點擊上放顯示模式按鈕，可以切換日間模式與深夜模式。
